# Welcome to Button | Component

This Component receive 4 Props with multiple types (disabledStatus, label, handleOnClick, buttonType) the prop label is mandatory.

This component can be easyly adapted changing the buttonType in between 'primary' or 'secondary' strings.

Or can be disabled through props disabledStatus.

And is prepared to handle a function on click.
